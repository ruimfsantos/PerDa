- Depois de fazer o clone:
  Compile o código. Será criada uma pasta \target.
  Copie os ficheiros .jar da pasta \target para a pasta SampleTest


- Comandos para executar:

  1) java -jar PerDa2Disco.jar --help

  2) java -jar PerDa2Disco.jar database-discovery -c

  3) java -jar PerDa2Disco.jar database-discovery -d

  4) java -jar PerDa2Disco.jar file-discovery
